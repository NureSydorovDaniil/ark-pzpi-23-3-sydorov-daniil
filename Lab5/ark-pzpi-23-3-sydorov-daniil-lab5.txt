	3 Опис виконаної роботи
	Під час виконання лабораторної роботи було виконано розгортання програмної системи для доставки пивної продукції. Система охоплює базу даних, верверну частину та IoT частину. Для перших етапів було спроєктовано загальну будову програмної системи для поставки пивної продукції, було визначено основи та їх взаємодію. Була розроблена база даних для проєкту «BeerDelivery» для її створення я використовував реляційну базу даних SQL Server Management Studio 22. Для роботи з базою даних було використоно мову програмування Python та бібліотеку SQLAlchemy. Було реалізовано прості функції для базових операцій з даними, на основі реалізованих функцій було створено сервірну логіку. У результаті виконання роботи було розроблено серверну частину програмної системи з доставки пивної продукції з базою даних та доступом до неї.
