2 Завдання
Методи рефакторингу коду програмного забеспечення

3 Опис Виконаної роботи
Рефакторинг коду є невід’ємною часстиною розробки програмного забезпечення. Рефакторинг позволяє покращити структуру системи без зміни її роботи. Головною метою під час виконання роботи було дослідити методи рефакторингу, які роблять код більш зрозумілим, легшим для подальшої роботи з ним й тестування.
Одним із головних методів рефакторингу який було розглянуто у презентації – метод виділення. Цей метод застосовується коли фрагменти коду можна згрупувати. Винисенн складної або довгої частини в окремий метод із підібраною назвою позволяє пояснити його призначення. Це зменшує довжину функції та спрощує її зрозумілість та сприйняття.
Важливим моментом у рефакторингу є усунення дублювання. Робота виконується для виявлення повторюваних фрагментів логіки та їх винесення до окремих класів. Використання цього методу дозволяє уникнути необхідності виносити зміни у багатьох частинах коду 	одночасно, що зменшу ризик винекнення помилок під час роботи.
Також важливий момент треба приділити методу перейменування змінних та методів. Використання коротких та незрозумілих назв погано вплияє на читабельність коду. Заміна їх на імена, що відповідають назві та описують суть процесу, допомагає вам та іншим розробникам швидше розуміти логіку коду без додаткових коментарів та поясснень.

4 Висновки
У ході виконання практичної роботи було розглянуто поняття рефакторингу коду та його головні методи, які спрямовані на покращенні структури коду. Дотримування методів при роботі з кодом дозволяє писати чистий та зрозумілий код з яким буде цікаво, просто та зручно працювати.

B.2 Код з методом виділення
def education_result(grades):
    total = 0
    for grade in grades:
        total += grade
    return total / len(grades)

def determine_status(average):
    if average >= 60:
        return "Зараховано"
    return "Не зараховано"

def show_student_result(name, grades):
    average = education_result(grades)
    status = determine_status(average)
    print("Студент:", name)
    print("Середній бал:", average)
    print("Статус:", status)

show_student_result("Студент 1", [60, 70, 90])

B.3 Код без методу усуненн дублювання
def student_info(name, score):
    if score >= 60:
        print(name)
        print("Результат Зараховано")
    else:
        print(name)
        print("Результат Не зараховано")

def exam_info(name, score):
    if score >= 60:
        print(name)
        print("Результат Зараховано")
    else:
        print(name)
        print("Результат Не зараховано")

student_info("Студент 1", 90)
exam_info("Студент 2", 59)

B.5 Код без методу переіменування змінних
def c(l):
    s = 0
    for i in l:
        if i % 2 == 0:
            s += i
    return s

lst = [10, 15, 20, 25, 30]
print(c(lst))


B.6 код з методом переіменування змінних
def numbers_sum(numbers_list):
    total = 0
    for number in numbers_list:
        if number % 2 == 0:
            total += number
    return total

my_numbers = [10, 15, 20, 25, 30]
print(numbers_sum(my_numbers))
